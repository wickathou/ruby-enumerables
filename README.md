# About The Project
## Ruby Enumerables

This project implements a custom take on multiple methods from the Enumerable's Module.

## Features:

### Built on Ruby

### Implementation of the methods
- each
- each_with_index
- select
- all?
- any?
- none?
- count
- map
- inject
- Implementation of the inject(:*) as multiply_els
